# booksAPI
My first node API! Enjoy!


## Setup

Just clone this repository and setup with command:

"npm start"

this will run locally in port 3001 or 3000

this api also is deployed in heroku

https://sheltered-mountain-66758.herokuapp.com/books

enjoy!
